## 🚀 Project Overview

This project is my first step into **Computational Fluid Dynamics (CFD)** 🌊
It demonstrates a **2D simulation of fluid flow** using the **Navier–Stokes equations** implemented in Python.

---

## ✨ Features

✔️ 2D Fluid Flow Simulation
✔️ Navier–Stokes Equation Implementation
✔️ Data Visualization 📊
✔️ Animated Fluid Motion 🎥
✔️ Clean & Beginner-Friendly Code
